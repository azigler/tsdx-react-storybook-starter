# HelloWorld

The `HelloWorld` component demonstrates `tsdx-react-storybook-starter` by loading and using the following file formats:

- GIF
- JPG
- PNG
- SVG
  - Component
  - URL
- WEBP
- CSS Modules
  - LESS
  - SASS
  - PostCSS
  - Stylus

You can test these different formats by changing the properties on `HelloWorld`.
